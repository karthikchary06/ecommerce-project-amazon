# 🛒 Ecommerce Project (Amazon Clone)

A full-stack ecommerce web application built using React, Express, and Sequelize.

## 🚀 Live Demo

🌐 Frontend (Vercel):  
https://ecommerce-project-amazon-nstpc3gn1-karthiks-projects-fa874fb0.vercel.app/

🔗 Backend (Render API):  
https://ecommerce-backend-x2gf.onrender.com/api/products

---

## 🏗 Tech Stack

### Frontend
- React (Vite)
- Axios
- React Router
- CSS

Deployed on: **Vercel**

### Backend
- Node.js
- Express.js
- Sequelize ORM
- SQLite (for demo)

Deployed on: **Render**

---

## 📦 Features

- 🛍 View Products
- 🛒 Add to Cart
- 💳 Checkout Page
- 📦 Orders Page
- 🚚 Delivery Options
- 💰 Payment Summary Calculation
- REST API Integration
- Full-stack deployment
